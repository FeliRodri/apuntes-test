## 📋 Descripción General

**Portfolio Web** es un sitio web personal para mostrar proyectos, habilidades y experiencia profesional. Diseñado con un enfoque en rendimiento y estética moderna.

## Objetivos

- ✅ Mostrar proyectos de desarrollo de forma atractiva
- ✅ Sitio rápido y optimizado (rendimiento 90+)
- ✅ Diseño responsive (móvil, tablet, desktop)
- ✅ SEO optimizado para visibilidad en buscadores
- ✅ Fácil de mantener y actualizar

## Tech Stack

| Tecnología | Uso |
|-----------|-----|
| **Astro** | Framework principal |
| **HTML/CSS** | Estructura y estilos |
| **JavaScript** | Interactividad |
| **Vercel** | Hosting y deploy |
| **Git** | Control de versiones |

## Características

### 🎨 Diseño
- Tema oscuro/claro
- Animaciones suaves
- Tipografía moderna
- Paleta de colores personalizada

### ⚡ Rendimiento
- Páginas estáticas pre-renderizadas
- Imágenes optimizadas
- CSS y JS minificados
- Lazy loading de componentes

### 📱 Responsive
- Mobile-first design
- Breakpoints para tablet y desktop
- Menú hamburguesa en móvil
